# Alzheimer_Disease_Detection

**Overview**
Developed a Convolutional Neural Network (CNN) model to detect and classify Alzheimer’s disease stages from MRI brain images.

**Tech Stack**
Python, TensorFlow/Keras, NumPy, Matplotlib, Scikit-learn

**Approach**
Image preprocessing & resizing
Data augmentation using ImageDataGenerator
CNN-based multi-class classification
Model evaluation using accuracy metrics

**Results**
Achieved reliable classification performance on MRI dataset using deep learning.
